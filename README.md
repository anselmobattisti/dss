# dss


## Installation

```shell
# create the venv environment
python3 -m venv venv

# access the venv
source venv/bin/activate

# install the packets
pip install -r requirements.txt

# install the component
pip3 install -e .

# install the SimPlacement
pip3 install -e ../SimPlacement

# leave the virtual environment
deactivate
```
